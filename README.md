DAT Project 28 - India Import Data Analysis

📌 Project Overview

This project analyzes India's import data from the last 10 years (2015-2025) to uncover trends in imported products, country-wise trade relationships, and financial insights. The data is structured for use in Power BI for visualization and reporting.

📂 Files in This Repository

DAT-project28.pdf - Summary of the Power BI dashboard with key insights.

DAT-project28.pbix - Power BI file containing interactive visualizations and dashboards.

India_Import_Data.xlsx - The structured dataset containing 3000 rows of India's import data.

📊 Data Description (India_Import_Data.xlsx)

The dataset consists of the following columns:

Sl. No - Serial number for reference.

Country - Name of the exporting country.

Region - Continent/region of the exporting country.

Product - Broad category of imported products.

Subcategory - Specific type of imported product within the main category.

Quantity - Quantity of imported goods.

Year - Year of import (2015-2025).

Month - Month of import in alphabetical format.

Amount - Total import amount in rupees (without decimals).

📈 Key Insights (From Power BI Analysis)

Top Imported Products: Major imports include Metals, Pharmaceuticals, Electronics, Automobiles, Agriculture, and Textiles.

Geographical Trends: Europe, Asia, and North America are the leading regions for India's imports.

Yearly Trends: Import amounts fluctuate across years, with noticeable peaks in certain years.

Seasonal Trends: Imports vary across months, indicating possible seasonal demand changes.

🛠 How to Use This Project

Power BI Users: Open DAT-project28.pbix in Power BI Desktop to explore interactive dashboards.

Excel Users: Open India_Import_Data.xlsx for manual data analysis.
